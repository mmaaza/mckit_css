Why I made McKit CSS?

I have used both Bootstrap and Tailwind CSS. Bootstrap CSS focuses on Components based approach while Tailwind CSS focuses on Utility First based approach. So I had an idea that what if I combine both of these apporaches, it will help the user in both ways; component as well utility wise approach.

-- Maaz Ahmad
